Инструкции по использованию FUDO PAM

Введение

FUDO PAM — это система контроля привилегированного доступа (Privileged Access Management), предназначенная для управления пользовательскими сессиями с возможностью контроля и автоматического реагирования на потенциально опасную активность в режиме реального времени.

Основные возможности

  • Активный мониторинг и запись сессий SSH, Telnet, RDP, HTTP, HTTPS, MySQL, Oracle, VNC и других.
  • Анализ архивных и активных соединений с возможностью контроля сеанса (прекращение, приостановка или присоединение администратором).
  • Интеграция с LDAP, Active Directory, системами авторизации (Wheel CERB, Radius и др.).
  • Модуль поведенческого анализа на основе машинного обучения для обнаружения подозрительной активности.
  • Управление паролями, включая хранилище и ротацию.
  • Поиск по сессиям с поддержкой OCR на русском, английском и других языках.
  • Генерация отчетов в PDF и периодические уведомления.

Начало работы

  1. Установка: Разверните FUDO PAM как шлюз (безагентное решение, настройка за 1 день).
  2. Добавление пользователей: Management → Users → добавить вручную или импортировать из AD/LDAP.
  3. Добавление целевых систем: Укажите серверы/приложения в разделе Targets.
  4. Настройка политик: Определите правила доступа и уведомления.
  5. Мониторинг: Активные сессии → вкладка OCR для поиска по тексту.
  6. Пароли: Настройте автоматическую ротацию.

Introduction

FUDO PAM is a Privileged Access Management (PAM) solution designed for managing user sessions with real-time control and automatic response to potentially dangerous activity.

Main Features

  • Active monitoring and recording of SSH, RDP, HTTP/HTTPS, databases, VNC etc.
  • Live session control (terminate, pause, join)
  • Integration with LDAP, Active Directory, Radius
  • ML-based behavioral analysis
  • Password vault & automated rotation
  • OCR search in sessions (including Russian)
