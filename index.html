Introduction

FUDO PAM is a Privileged Access Management (PAM) solution designed for managing user sessions with real-time control and automatic response to potentially dangerous activity.

Key Features

  • Active monitoring and recording of SSH, Telnet, RDP, HTTP, HTTPS, MySQL, Oracle, VNC sessions and more.
  • Real-time session control (terminate, pause, join).
  • Integration with LDAP, Active Directory, Radius, etc.
  • Behavioral analysis module based on machine learning.
  • Password management and rotation.
  • OCR search in sessions (Russian & English).
